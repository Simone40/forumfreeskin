Il nuovo SkinParser
Notifica nuove risposte | Invia tramite email | Stampa Discussione
Inviato il: 14/8/2009, 14:11 da: Hannibal_Smith
Il tempo è mancato, cssutils purtroppo è stato meno utile del previsto per via di alcuni suoi bug e mi è toccato riscrivere quello a cui stavo lavorando.
Ecco a voi il nuovo SkinParser (Il prima e dopo)
Oltre il codice che è nettamente migliorato e che necessità di Python 2.7, supporta il parsing del css del tipo:
CODE
selettore {
proprietà: valore;
}

E restituisce css formattato del tipo:
CODE
selettore {
   proprietà: valore;
}

Modalità Modifica Forzata
Notifica nuove risposte | Invia tramite email | Stampa Discussione
Inviato il: 19/3/2009, 16:04 da: Hannibal_Smith
In questi giorni ha lavorato poco a FFS, ma c'è una bella novità: La modifica forzata

Non è sempre facile trovare una corrispondenza esatta tra css ed elementi html. La modalità forzata permette di superare questa difficoltà. Quando si seleziona un elemento e FFS non è in grado di trovare la corrispondenza nel css, nella modalità forzata FFS ridefinisce il selettore non trovato .
Segnalo anche che SkinMap è stata quasi completamente eliminata

Spero per la prossima settimana di inserire la modifica dei bordi
